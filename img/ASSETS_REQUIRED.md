# Assets pendientes — carpeta `/img`

El código ya apunta a estas rutas exactas. Solo sube los archivos con estos nombres y se conectarán automáticamente (no hay que tocar HTML/CSS).

| Archivo esperado              | Dónde se usa                        | Tamaño recomendado |
|--------------------------------|--------------------------------------|---------------------|
| `logo-emmanuel.png`            | Navbar (fondo oscuro)                | 128×128 px, fondo transparente |
| `logo-emmanuel-white.png`      | Footer                               | 128×128 px, fondo transparente |
| `hero-background.jpg`          | Fondo del Hero (Dawn)                | 2400×1600 px aprox., escena real y humana, nunca clichés religiosos |
| `about-community.jpg`          | Sección "Acerca de Emmanuel"         | 1000×1250 px (retrato 4:5) |
| `devo-01-luz.jpg`               | Card devocional "La luz que no se apaga" | 800×500 px |
| `devo-02-silencio.jpg`          | Card devocional "Cuando Dios parece silencioso" | 800×500 px |
| `devo-03-presencia.jpg`         | Card devocional "Pequeños actos, presencia grande" | 800×500 px |
| `og-cover.jpg`                  | Vista previa al compartir en redes (Open Graph) | 1200×630 px |

Mientras no subas las imágenes, esos espacios se ven como fondos degradados suaves (azul/dorado) — no como un ícono de imagen rota — así que el sitio nunca se ve "incompleto" aunque falten fotos reales.

Sigue la Photography Direction del Volumen I: momentos reales, no sonrisas fingidas ni "staged worship".
